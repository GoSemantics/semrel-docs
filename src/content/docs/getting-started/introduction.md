# Welcome to go-semrel

**go-semrel** is a powerful, semantic versioning tool for Go projects. It automatically determines the next version based on your commits and can manage versioning in monorepos.

## Features

- **Automated Versioning**: Analyzes commit messages to determine semantic versions
- **Monorepo Support**: Independent versioning for multiple projects in a single repository
- **Plugin System**: Extend functionality with custom plugins
- **gRPC API**: Programmatic access to versioning logic
- **Language Agnostic**: Use semver logic from any language via gRPC
