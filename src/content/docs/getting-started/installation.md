# Installation

## From Binary

Download the latest release from [GitHub Releases](https://github.com/GoSemantics/go-semrel/releases).

## From Source

```bash
go install github.com/GoSemantics/go-semrel/cmd/go-semrel@latest
```

## Verify Installation

```bash
go-semrel version
```
