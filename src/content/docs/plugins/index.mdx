Official semrel plugins are published as standalone binaries named `semrel-plugin-<name>`.

Install any official plugin with:

```bash
semrel plugin install <name>
```

For example, `semrel plugin install github` installs the `semrel-plugin-github` binary into `~/.semrel/plugins/` by default.

## Discovery and installation

- Default install directory: `~/.semrel/plugins/`
- Binary naming convention: `semrel-plugin-<name>`
- Install command: `semrel plugin install <name>`
- Search command: `semrel plugin search <query>`
- List installed plugins: `semrel plugin list`

See the [Plugin Registry](/plugins/registry/) for registry metadata and installation details.
