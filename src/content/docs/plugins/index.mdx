# Official Plugins

SemRel ships with a comprehensive plugin system. Here are the official plugins maintained by the SemRel team.

## Available Plugins

### Providers
- **provider-github** - GitHub release provider
- **provider-gitlab** - GitLab release provider
- **provider-gitea** - Gitea release provider  
- **provider-git** - Git CLI provider (bare git, works with any forge)

### CI Conditions
- **condition-github-actions** - GitHub Actions environment checks
- **condition-gitlab-ci** - GitLab CI environment checks
- **condition-gitea-actions** - Gitea Actions environment checks
- **condition-generic** - Generic CI env var assertions

### Analyzers
- **analyzer-conventional** - Conventional Commits analyzer
- **analyzer-default** - Default SemVer analyzer

### Generators
- **generator-changelog-md** - Markdown changelog generator
- **generator-changelog-html** - HTML changelog generator
- **generator-release-notes** - Release notes generator

### Updaters (Version Managers)
- **updater-npm** - Update package.json versions
- **updater-python** - Update Python package versions
- **updater-gradle** - Update Gradle versions
- **updater-docker** - Build and publish Docker images
- **updater-go** - Update Go module versions

### Hooks (Notifications)
- **hook-slack** - Send Slack notifications
- **hook-email** - Send email notifications (Phase 2)

## Registry

All official plugins are published to the [Plugin Registry](/plugins/registry/).

**Registry URL:** https://registry.semrel.io

## Discovery

Plugins are auto-discovered from:
1. Registry: `https://registry.semrel.io/plugins.json`
2. Local cache: `.semrel/plugins/`
3. Direct installation (copy binary to `.semrel/plugins/`)

See [Registry documentation](/plugins/registry/) for details.
