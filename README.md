# semrel-docs

Documentation & website for semrel built with Astro and Starlight.

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to preview.

## Build

```bash
npm run build
```

## Deployment

Deploys to GitHub Pages at `semrel.io` via GitHub Actions.
